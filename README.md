## How to Use

1. Open `index5.html` in your web browser
2. Click on any song from the playlist to play
3. Use the control buttons to manage playback
4. Adjust volume using the volume slider
5. Browse through popular songs and artists

## Setup

1. Clone this repository
2. Make sure all audio files are in the `audio/` folder
3. Ensure all images are in the `Image/` folder
4. Open `index5.html` in a modern web browser
